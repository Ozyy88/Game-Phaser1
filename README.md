# Coin Hunter Game

Coin Hunter adalah game platformer 2D sederhana yang dikembangkan dengan Phaser 3. Pemain harus mengumpulkan koin, melompat dari platform ke platform, dan menghindari musuh yang muncul saat level meningkat.

## Cara Bermain
- Buka `index.html` di browser.
- Tekan tombol `Play` untuk memulai.
- Gunakan tombol `←` / `→` atau `A` / `D` untuk bergerak.
- Gunakan tombol `↑`, `W`, atau `Space` untuk melompat.
- Kumpulkan semua koin untuk naik ke level berikutnya.
- Hati-hati dengan musuh: terkena musuh berarti permainan selesai.

## Fitur
- Level bertahap dengan tujuan mengumpulkan koin.
- Sistem double jump.
- Animasi karakter bergerak, melompat, dan berdiri.
- Musik latar dan efek suara.
- Tampilan game yang responsif dan overlay orientasi pada perangkat potret.

## Struktur Proyek
- `index.html` – titik masuk halaman game.
- `scenePlay.js` – logika permainan utama.
- `Assets/` – grafik dan audio untuk game.

## Teknologi
- Phaser 3
- HTML
- CSS
